## Features

<CardGrid stagger>

		Update `--brand-h` (0–360) and your entire palette shifts: backgrounds, borders, text, and accents.
	</Card>
	<Card title="OKLCH Color Space" icon="sun">
		Perceptually uniform, wide-gamut colors that look great at any hue.
	</Card>
	<Card title="Optional Widget" icon="puzzle">
		Includes an optional **HuePicker** widget — a floating circular color wheel for live hue selection.
	</Card>
	<Card title="Tailwind CSS v4" icon="rocket">
		Built for the modern Tailwind core. Zero runtime dependencies.
	</Card>
